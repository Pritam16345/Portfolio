CLOAKENT

ENTERPRISE AI DATA FIREWALL

A production Zero-Trust DLP gateway that stands between your enterprise and every LLM API call. It detects and redacts 15+ PII and PHI entity types — names, emails, SSNs, medical codes — in under 200ms, before the model ever sees them.

The Bidirectional Anonymization Protocol reverses this process on the LLM's response — restoring entities contextually, maintaining data sovereignty without breaking the user experience. Audit logs via SQLAlchemy. Deployed on Hugging Face Spaces and Vercel via Docker microservices.
